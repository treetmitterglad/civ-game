# Civ-Game

A civilization-building game developed in Godot Engine, featuring a top-down orthographic view and 64x64 pixel art assets.

---

## Description

Civ-Game is a strategy game where players build and manage their own civilization. Inspired by classic civilization-building games, it offers a unique blend of resource management, exploration, and expansion in a pixel art style.

---

## Features

- **Top-Down Orthographic View**: Navigate and manage your civilization from a clear, isometric perspective.
- **64x64 Pixel Art Assets**: Visually appealing and retro-styled graphics for an immersive experience.
- **Resource Management**: Gather, trade, and utilize resources to grow your civilization.
- **Exploration and Expansion**: Discover new territories and expand your empire.
- **Godot Engine**: Built using the powerful and flexible Godot Engine for smooth gameplay and easy modification.

---

## Installation

1. **Prerequisites**:
  - [Godot Engine 4.x](https://godotengine.org/) (or the version used for development)
  - Basic knowledge of Godot's interface and scripting (GDScript)
2. **Steps**:
  - Clone or download the repository to your local machine.
  - Open the project in Godot Engine.
  - Run the menu scene (`main_menu`) to start the game.
